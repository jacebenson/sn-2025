# Debugging a widget
A "lifesaver" [Travis Toulson shared](https://www.linkedin.com/events/livecodingafternoontea-withtrav7359215939516604418/theater/), inspect an element on a widget (while on the portal page), then go to the console, type out `angular.element($0)`

This grabs the current selected element and shows you all the variables in the client controller

[Docs](https://www.servicenow.com/docs/csh?topicname=service-portal-widgets.html&version=latest)
